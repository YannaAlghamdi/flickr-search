# flickr-search

An Angular app for searching photos on Flickr using Flickr Search API

## Technologies Used

- Angular 11
- Ionic 5

## Getting Started

- Before running the UI, make sure that the following tools are installed in your machine:
  - NPM (v6 or higher)
  - Node (v10 or higher)
  - Ionic (v5)
- To run this project, go to the root directory (`/flickr-search`) and execute the following commands:

```bash
$ npm install -g @ionic/cli //if you need to install Ionic CLI
$ npm install
$ ionic serve
```

- Browse thru localhost:8100 on your favorite browser. Enjoy! :)
